Comandos

  • Criar um usuarios:

    git config --global user.name "qualque nome"

  • Criando email:

    git config --global user.email"email@mail.com

  • Criar uma pasta:

    mkdir nomepasta

  • Cntra na pasta e iniciar o git:

    cd nomedapasta git init

  • Configurando um editor de texto: sublime-text

    git config --global core.editor subl

  • verificando o estatos

    git status

  • Arquivos quando for modificado(Modifiel)

    git add nomeArquivo

  • Quando esta preparado pro commit (staged):

    git commit -am "descrição"
